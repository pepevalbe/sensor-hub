Sensor de Temperatura y Humedad

Para este dispositivo nos basamos en el ESP-8266. Se trata de un módulo WiFi de bajo coste ampliamente utilizado en multitud de proyectos basados en Arduino. Se puede utilizar junto con otro microcontrolador al que aportaría conectividad WiFi o programarlo directamente para cualquier aplicación. El fabricante Wemos dispone de modelos compactos al que se pueden apilar sensores.

Componentes:
  • Wemos D1 Mini Lite
  • Wemos SHT30 Shield

El montaje es muy sencilo simplemente hay que soldar a través de unos pines el módulo SHT30 a la placa D1 Mini Lite. Además se unen a través de un jumper los pines D0 y RST de forma que se puedan desconectar. Para cargar programas a tráves del conector micro usb el pin RST debe estar libre.

Software:
  • Instalar ESP8266 al entorno Arduino

    Para la programación podemos utilizar el IDE de Arduino gracias a las librerias desarrolladas por la comunidad de ESP8266. Una vez instalado el plugin elegimos la placa "WeMos D1, R2 & Mini" para poder compilar y subir nuestro código.

  • Librería Wemos SHT30

    El sensor SHT30 se maneja a través de un puerto I2C. Hay disponible una librería del fabricante para facilitar su uso. Con ella nos abstraemos de la comunicación a bajo nivel y ya tiene en cuenta el conexionado de los pines por lo que no requiere configuración. Para utilizarla basta con copiarla a la carpeta libraries de Arduino.

  • Sketch de ejemplo

    Tras estos pasos ya podemos cargar el sketch. Tan solo hay que configurar los siguentes parámetros: ssid, password y token. El programa es simple, una vez se alimenta a través del conector micro usb lo primero que hace es obtener la temperatura y humedad. A continuación se conecta a la red wifi y hace una llamada http post al servidor. Finalmente se pone en modo reposo para no consumir y evitar que suba la temperatura del circuito unos grados debido al funcionamiento del WiFi. Tras 20 minutos se produce una interrupción que reinicia la placa y vuelve a comenzar (es necesario que los pines D0 y RST estén conectados).
