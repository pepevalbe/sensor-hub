Registro de accesos

Para controlar la apertura de una puerta utilizaremos un switch magnético y un imán de tal modo que cuando se mueva el imán cuando se abre la puerta se produzca una interrupción en una placa Arduino. El envío de datos (una única llamada para que se registre la hora) lo realizamos con un módulo SIM800L, se trata de un dispositivo GSM/GPRS de bajo coste con el que con una tarjeta SIM se pueden hacer llamadas web.

Componentes:
  • Arduino Nano
  • SIMCOM SIM800L
  • Switch Magnético

Para el montaje debemos conectar al microcontrolador el módulo SIMCOM (pines TX/RX y RST) y el switch magnético (pin de interrupción). Alimentamos tanto la placa Arduino como el SIM800L con una batería de litio de una célula (4.2v). Para que funcione debemos fijar el conjunto en el marco de la puerta y colocar con cuidado un imán en el borde, de tal forma que quede enfrentado al switch magnéntico

Software:
  • Librería SIMCOM 800L

    El módulo SIMCOM se maneja desde un microcontrolador a través de comandos AT. Con esta librería nos ahorramos el trabajo de bajo nivel y podemos controlar el módulo a través de una clase sencilla en Arduino. Para utilizarla basta con copiarla a la carpeta libraries de Arduino.
