Hadware

Cualquier tipo de dispositivo con sensores que se pueda programar para hacer llamadas HTTP se puede integrar sin problema en esta aplicación. Es especialmente fácil con Arduino pues hay multitud de microcontroladores, módulos de conectividad (WIFI, GSM/GPRS) y sensores compatibles a bajo coste. Además cuenta con una gran comunidad, hay librerías disponibles para casi todo por lo que la programación es muy sencilla. A continuación se muestran algunos proyectos de ejemplo, basados en Arduino, para construir tu propios dispositivos.
